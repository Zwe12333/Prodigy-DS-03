# E-commerce Purchase Prediction using Amazon Reviews
**Prodigy infotech internship project| Hlangalezwe Zulu | Nov 2025

## Project Overview
Built a Decision Tree Classifier to predict whether a customer will purchase based on Amazon review data.
- Real dataset (amazon.csv with 140k+ rows)
- Full ML pipeline: Upload data → EDA → Preprocessing → Modeling → Evaluation
- Key insights: Duration, month, previous outcome drive purchases
- Achieved strong ROC-AUC and interpretable results
